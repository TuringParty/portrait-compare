# 人脸识别算法说明

## 方案A：YuNet + SFace（推荐）

### 人脸检测：YuNet
- **模型**：`face_detection_yunet_2023mar.onnx`
- **特点**：轻量级、高速、支持多尺度人脸检测
- **输出**：5个关键点（左右眼、鼻子、左右嘴角）+ 置信度分数
- **推荐场景**：图片质量较好、需要精确定位

### 人脸识别：SFace
- **模型**：`face_recognition_sface_2021dec.onnx`
- **特点**：基于 Sigmoid 函数的约束软最大间隔损失训练
- **特征维度**：128 维
- **官方推荐阈值**：余弦相似度 ≥ 0.593（同一人）
- **论文**：SFace: Sigmoid-Constrained Hypersphere Loss for Robust Face Recognition

## 方案B：Haar Cascade + HOG（通用）

### 人脸检测：Haar Cascade
- **模型**：OpenCV 内置 `haarcascade_frontalface_default.xml`
- **特点**：经典算法，速度快，仅支持正面人脸
- **局限**：侧脸、遮挡、角度偏大时效果较差

### 特征提取：HOG + 分块直方图
- **HOG（方向梯度直方图）**：捕捉脸部形状和纹理信息
  - 输入尺寸：128×128
  - 块大小：16×16，步长：8×8
  - 方向 bin 数：9
  - 特征维度：8100 维
- **分块颜色直方图**：4×4 网格，每格 16 bin
  - 特征维度：256 维
- **总特征维度**：8356 维（L2 归一化）

## 相似度计算

```
余弦相似度 = 1 - cosine_distance(feat1, feat2)
相似度分数 = (余弦相似度 + 1) / 2  ∈ [0, 1]
```

## 影响识别精度的因素

| 因素 | 影响程度 | 说明 |
|------|---------|------|
| 光照变化 | ★★★★☆ | 强烈光照差异会降低匹配率 |
| 拍摄角度 | ★★★★★ | 侧脸 vs 正脸误差大 |
| 图片分辨率 | ★★★☆☆ | 建议人脸区域 ≥ 80×80 像素 |
| 表情变化 | ★★★☆☆ | 夸张表情（大笑/哭泣）影响较大 |
| 年龄跨度 | ★★★☆☆ | 同一人跨越10年以上照片匹配率下降 |
| 妆容/眼镜 | ★★☆☆☆ | 轻度影响 |
